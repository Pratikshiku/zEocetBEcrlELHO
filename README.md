## Agricultural Product Direct Sales Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/55a2b7c731b8483d9a1c32628da3fa4e/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, merchants, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Announcement Information
5. Agricultural Products
6. Merchants
7. Personal Center
8. Delivery Address
9. Agricultural Product Favorites
10. Agricultural Product Orders
11. Background Management
12. Shopping Cart




**Merchant Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Announcement Information
6. Agricultural Product Management
7. Agricultural Product Review Management
8. Agricultural Product Order Management



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Announcement Type Management, Agricultural Product Type Management, Merchant Credit Type Management)
4. Announcement Information Management
5. Agricultural Product Management
6. Agricultural Product Review Management
7. Agricultural Product Collection Management
8. Agricultural Product Order Management
9. Merchant Management
10. User Management
11. Carousel Management
